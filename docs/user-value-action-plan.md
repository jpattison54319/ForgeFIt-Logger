# ForgeFit User-Value Action Plan

_Date: 2026-07-09 · Status: approved · Companion to [product-growth-plan.md](product-growth-plan.md) (strategy layer). This document is the **evidence-verified execution layer**: every current-state claim is anchored to the working tree (branch `feature/exercise-modality-and-intervals`, audit of 2026-07-08/09), every market claim carries a source. Appendices hold the full evidence._

## How this plan was built

1. **Four deep codebase audits** — onboarding/Home, strength loop, cardio/yoga/import, recovery/analytics/retention — with file:line evidence for every claim (Appendix A).
2. **The founder's own Hevy account** — 439 workouts, ~50 routines: ground truth for what the target persona actually uses (Appendix D).
3. **Live competitor intelligence, July 2026** — pricing, 2025–26 moves, documented complaints for Hevy, Strong, Strava, Down Dog, Athlytic, Bevel, Whoop, Gentler Streak + threat scan (Appendix B).
4. **Retention science** — Duolingo's published streak data, habit RCTs, permission-priming and retention benchmarks (Appendix C).
5. **Build health** — `make test` green (ForgeCore/ForgeData suites + stub package builds).

## The verdict

The audits agree: **ForgeFit's engines are already excellent — several beat the competition today** (ghost-value set logging, readiness math with nocturnal HRV windows and honest "building" states, a coach button that actually mutates the started workout, Live Activity with rest takeover, bulletproof session restore, five-source import machinery). What loses users is **delivery and flow**: an empty first-run for anyone who skips the optional program, HealthKit demanded before value, a notification system that's silently off for everyone who doesn't find one Settings card, a readiness score that's stale by morning because nothing refreshes in background, one genuine trust bug (a zone chart labeled "Measured" showing estimated data), and the flagship "progression coaching" claim with **no progression engine behind it** (`progressionRuleID` is a dead field).

Meanwhile the market window is open: Strava is burning goodwill with paywall creep (Year in Sport paywalled Dec 2025), Whoop torched trust with upgrade fees + an FDA warning letter, Bevel tripled Pro to $99.99/yr, Strong is coasting, Hevy is documented as "logs but doesn't coach," Athlytic's #1 dropout reason is "no workout programming built in" (the exact loop ForgeFit already closes), and Bloomberg reports **Apple scaled back its AI health coach in Feb 2026** — Apple just conceded this lane to third parties.

# Appendix D — Founder's Hevy profile (the persona, from the live API)

439 workouts · ~50 routines in 5 folder pages · themed program folders ("Thor: God of Gains") + Upper/Lower split · supersets used (Upper A/B, 2 groups each) · set types in real use: normal, warmup, dropset, failure · **RPE never used** · per-exercise rest timers 90–120s in newer routines, 0 in older · custom machine-specific exercises everywhere ("NewTech Assisted Dip," "Atlantis Horizontal Bicep Isolator," "MTS Incline press") · weights are pounds stored as noisy kg floats (110 lb = 49.895220037920375 kg) · sessions 4–6 exercises, 2–4 sets, 26–43 min · same-weight-across-sets is the norm (double-progression behavior) · **zero cardio logged in Hevy** — the fragmentation thesis in one account.
